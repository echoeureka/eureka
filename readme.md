# cbgcli

Some features to smplify my config or any other that can be handled automatically.

features
  - git proxy
  - git unset proxy
  - git user config
  - VitePress Template

how to use it?

```
star this repo
fork this repo
```

Change user object -> user.name, user.email in index.js.
You can just yarn create or npm init cbgcli to use other feactures.
